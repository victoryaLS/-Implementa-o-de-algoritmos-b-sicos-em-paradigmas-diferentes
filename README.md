# -Implementacao-de-algoritmos-basicos-em-paradigmas-diferentes
Comparação entre os Paradigmas

Na minha opinião, o paradigma mais fácil de entender foi o imperativo, porque ele mostra passo a passo o que o programa está fazendo. Basta usar um loop e algumas variáveis para chegar ao resultado. O paradigma orientado a objetos é mais organizado, pois cria classes para representar os funcionários e separar as responsabilidades do sistema, mas acaba ficando um pouco maior. Já o paradigma funcional usa funções como filter, map e reduce, o que deixa o código mais curto, porém pode ser um pouco mais difícil de entender no começo. Para esse problema simples, eu achei o imperativo mais intuitivo.
